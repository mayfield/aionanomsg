aionanomsg
========
Asyncio nanomsg library.


About
--------
Layers on top of, https://github.com/nanomsg/nnpy.git


Requirements
--------
* nnpy Python Library
    * cffi (>=1.8.3)
    * nanomsg (>=1.0)


Installation
--------
    python3 ./setup.py build
    python3 ./setup.py install


Compatibility
--------
* Python 3.5+
